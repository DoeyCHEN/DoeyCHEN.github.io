Hexo主要命令

  • 新建文章: hexo new “postName” 在 source/_posts/ 下生成 postName.md
  • 新建页面: hexo new page “pageName” 在 source/ 下生成 pageName 文件夹
  • 清理: hexo clean 清理 db.json 和 public 文件夹
  • 生成静态文件: hexo g 静态文件在 public 文件夹下
  • 本地预览: hexo s 打开 http://localhost:4000 进行本地预览
  • 部署项目: hexo d 部署至服务器

Markdown语法简要规则

列表

在 Markdown 下
列表的显示只需要在文字前加上 - 或 * 即可变为无序列表
有序列表则直接在文字前加 1. 2. 3. 符号要和文字之间加上一个字符的空格。

引用

> 例如这样
只需要在文本前加入 > 这种尖括号（大于号）即可

图片与链接

插入链接与插入图片的语法很像，区别在一个 ! 号

插入链接 [Baidu](http://baidu.com)
插入图片 ![Pics](http://github.com/...png)

粗体与斜体

Markdown 的粗体和斜体也非常简单，用两个 * 包含一段文本就是粗体的语法，用一个 * 包含一段文本就是斜体的语法。

例如：这里是粗体 这里是斜体

分割线

分割线的语法只需要另起一行，连续输入三个星号 *** 即可。
