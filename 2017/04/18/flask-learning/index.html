将route()装饰器绑定到对应url上

变量规则

为url添加 变量部分 <variable_name>
可以用 <converter:variable_name>

其中 converter有
int(接受整数);float(接受浮点数);path(和默认的相似，但也接受斜线)

唯一url和重定向

构造url

需import url_for模块
使用url_for()关键字为函数指定url
它接受函数名作为第一个参数，也接受对应url规则的变量部分的命名参数。

规定http方法

默认情况下，如@app.route('/')只回应get请求
可以通过route()装饰器传递methods

http方法有get, head, post, put, delete, options

静态文件

通常为css js文件，放在static文件夹中，使用／static访问
给静态文件生成url

模版渲染

使用render_template()方法来渲染模板。你需要做的一切就是将模板名和你想作为关键字的参数传入模板的变量。
